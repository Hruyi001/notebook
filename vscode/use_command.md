## 三、快捷键
* Ctrl + P: 转到文件/其他常用操作
* 打开命令面板: Ctrl + Shift + P
* 关闭当前文件：Ctrl + W
* 当前行上移/下移： Alt + Up/Down
* 打开终端： Ctrl + `
* 关闭侧边栏：Ctrl + B
* 变量统一重命名：F2
* 转到定义处：F12
* 全屏： F11
